# DeepSeek-V4-Flash 深度解析

本项目包含 DeepSeek-V4-Flash 模型的源码解析和交互式学习课程。

## 📁 项目结构

```
.
├── DeepSeek-V4-Flash/       # DeepSeek-V4-Flash 模型源码
├── deepseek-v4-course/      # 交互式学习课程
└── DeepSeek-V4-Flash-源码解析.md  # 源码解析文档
```

## 🎯 课程内容

交互式课程包含 6 个模块：

1. **项目概览** - 了解 DeepSeek-V4-Flash 的核心特性
2. **Transformer 核心架构** - 深入理解模型结构设计
3. **文本生成推理流程** - 理解模型如何生成文本
4. **编码系统详解** - 探索消息编码和分布式词汇表
5. **高效计算技术** - 学习量化和性能优化
6. **实战总结与部署** - 学习如何部署使用

## 🚀 运行课程

直接在浏览器中打开 `deepseek-v4-course/index.html` 即可开始学习，无需服务器。
